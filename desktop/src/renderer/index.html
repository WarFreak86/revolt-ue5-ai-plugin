No telemetry

          
Offline Mode

          
The desktop runtime launches without login, cloud providers, telemetry, or model inference.

        

        

          
    
            
  • No cloud provider is configured by default.
    • 
            
  • No telemetry collection is implemented.
    • 
            
  • No model inference is implemented in Phase 7.
    • 
            
  • Phase 8 model routing supports local/offline model endpoints only.
    • 
            
  • Phase 9 indexing stores files, chunks, asset metadata, and scene summaries locally.
    • 
            
  • Unreal communication uses the configured local bridge endpoint.
    • 
            
  • With no model selected, the app remains in Manual Command Mode.
    • 
            
  • Dangerous permission mode is disabled unless explicitly enabled in config.

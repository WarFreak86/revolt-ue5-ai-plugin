# Known Limitations

- Generated Unreal assets are created when bridge commands are run in Unreal, not during repository checkout.
- Desktop packaging is a local zip package, not a signed platform installer.
- MCP tools expose a focused subset of bridge commands.
- Local model quality depends on the user-provided local model server.
- Multiplayer systems are marked as assumptions/placeholders unless explicitly implemented.
- Save/load hooks in gameplay templates are placeholders.
- Online providers are not required and are disabled by default.
